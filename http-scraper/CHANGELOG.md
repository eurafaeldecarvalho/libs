# Changelog

## 0.1.0

- Initial Node.js and TypeScript port of `hrequests` 0.9.2.
- Official `hrequests-cgo` bridge integration for TLS-fingerprinted HTTP requests.
- Requests-style sessions, cookies, HTML parsing, async request helpers, and concurrent request mapping.
- Built-in browser rendering layer with `playwright-core` and local Chrome/Chromium or Firefox executables.
- npm release scripts, automated tests, and packaging validation.
